# webMusic


## 运行项目
```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

```
## 后端接口

```
https://binaryify.github.io/NeteaseCloudMusicApi/#/?id=neteasecloudmusicapi
```

## 技术栈

`vue2 + vuex + vue-router + ES6/7 + axios + elementUI + 阿里图标iconfont`
